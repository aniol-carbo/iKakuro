Per a consultar la documentació automàtica de codi generada per javadoc, s'ha d'obrir la carpeta "javadoc" i fer doble clic sobre el fitxer "index". S'obrirà una pestanya al navegador, que permetrà consultar tota aquesta documentació de forma interactiva.

El fitxer "Jocs de prova" conté una llista dels passos a seguir per a provar les funcionalitats principals de l'aplicació.

El Manual d'Usuari conté les instruccions de l'aplicació, explicant totes les funcionalitats que ofereix.

El fitxer "Tests d'Integració" conté la demostració del funcionament correcte de totes les funcionalitats de l'aplicació, amb els cursos alternatius que es poden donar en tots els escenaris.

El fitxer "Relació classes Implementades" mostra les classes que ha implementat cada integrant de l'equip.
